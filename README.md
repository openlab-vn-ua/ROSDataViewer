OVERVIEW
-----------------------
App for reading files of any size / navigating on them.

*.BAG files get from https://github.com/udacity/self-driving-car/tree/master/datasets
(converted to *.CSV)


System requirements:
* Windows XP/7/8/10.
* Ubuntu (didn't testing).
* Min screen resolution 1024x768.
